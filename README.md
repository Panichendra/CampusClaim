# CampusClaim – Centralized Lost & Found Management System

CampusClaim is a full-stack platform designed to modernize and organize the lost-and-found process within the university campus. It replaces fragmented communication channels with a secure, real-time digital inventory for reporting and recovering lost items.

live link: https://campus-claim-frontend-nine.vercel.app/
---

## 🛠️ Technical Implementation

* **MERN Stack Architecture**: Developed a scalable web application using **MongoDB**, **Express.js**, **React.js**, and **Node.js**.
* **Secure Authentication**: Implemented **JSON Web Tokens (JWT)** for robust user authentication and secure session management.
* **Image Processing**: Integrated **Multer** middleware to handle multipart form data, enabling users to upload photos of found items for visual verification.
* **RESTful API Services**: Engineered efficient backend endpoints for item reporting, claim processing, and automated status updates.
* **Responsive Frontend**: Designed a mobile-first, component-based user interface in **React** to ensure seamless accessibility for users across various devices.

---

## 🎯 Core Use Cases

* **Campus-Wide Reporting**: Provides a centralized database for reporting lost or found items, ensuring visibility for all campus members.
* **Visual Verification**: Allows finders to provide photographic evidence, helping owners identify their belongings remotely before initiating a claim.
* **Secured Claiming Process**: Requires users to be authenticated via the campus system to submit claims, reducing the risk of unauthorized possession.
* **Categorized Search**: Enables users to filter and search through item lists by category, date, and location to find specific misplaced belongings quickly.
